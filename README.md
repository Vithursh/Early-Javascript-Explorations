# Early JavaScript & Creative Coding Archive (2020 – 2021)

This repository is a curated collection of over 10+ sketches and experiments created during my early journey into software development. I built these using **p5.js** to master the fundamentals of JavaScript, logic, and computer graphics.

## 🚀 Why this exists
I started coding as a teenager, and these projects represent my "building blocks." While the code reflects my early learning stages, these projects taught me:
*   **Core JS Logic:** Loops, conditionals, and functional programming.
*   **Mathematical Concepts:** Coordinate systems, vectors, and collision detection.
*   **Creative Problem Solving:** Turning visual ideas into interactive code.

## 📂 What's Inside?
This archive includes various experiments, such as:
*   **Algorithmic Art:** Generative patterns and shapes.
*   **Interactive Sketches:** Mouse and keyboard-responsive visuals.
*   **Game Prototypes:** Early attempts at physics and game loops.

## 🛠️ Tech Stack
*   **Language:** JavaScript (ES5/ES6)
*   **Library:** [p5.js](https://p5js.org)
